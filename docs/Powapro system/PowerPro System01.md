## ①〜⑦ 定義（最終版）

| 番号 | 内容                                 |
| ---- | ------------------------------------ |
| ①    | 能力（フィジカル・判断・役割遂行力） |
| ②    | テクニック（動作・ボール扱い）       |
| ③    | シュートテクニック                   |
| ④    | よくある弱点                         |
| ⑤    | ハンドボールプレーヤーとしての強み   |
| ⑥    | 各ポジションに求められる役割         |
| ⑦    | 各ポジションがやってはいけない勘違い |

------

# ウイング（LW / RW）

| 項目         | 内容                                                   |
| ------------ | ------------------------------------------------------ |
| ① 能力       | スピード、加速力、空間認知、速攻参加力                 |
| ② テクニック | 片手キャッチ、ライン際ステップ、空中姿勢制御           |
| ③ シュート   | 角度シュート、ローリング、ワンバウンド                 |
| ④ 弱点       | 体勢不安定、守備1on1不足、決定率の波                   |
| ⑤ 強み       | 狭角からの得点力、速攻で流れを変える                   |
| ⑥ 役割       | サイドで幅を作る、速攻のフィニッシャー                 |
| ⑦ 勘違い     | 「点を取れれば守備は不要」「待っていればボールが来る」 |

------

# バック（LB / RB）

| 項目         | 内容                                     |
| ------------ | ---------------------------------------- |
| ① 能力       | フィジカル、ジャンプ力、ロング対応力     |
| ② テクニック | ジャンプ動作、ステップバック、展開力     |
| ③ シュート   | ロング・ジャンプ・スピン                 |
| ④ 弱点       | 精度ムラ、リリース遅れ、戻り遅い         |
| ⑤ 強み       | ミドルから守備を下げられる存在           |
| ⑥ 役割       | 得点圧で守備を収縮させる                 |
| ⑦ 勘違い     | 「強く投げれば入る」「常に打つのが仕事」 |

------

# センターバック（CB）

| 項目         | 内容                                     |
| ------------ | ---------------------------------------- |
| ① 能力       | 判断力、視野、ゲームコントロール         |
| ② テクニック | クイックパス、テンポ調整                 |
| ③ シュート   | クイック、コース狙い                     |
| ④ 弱点       | 判断遅れ、消極性、中央管理不足           |
| ⑤ 強み       | 味方を活かし流れを支配                   |
| ⑥ 役割       | 攻撃の設計・判断の中心                   |
| ⑦ 勘違い     | 「自分が一番上手い」「全部自分で決める」 |

------

# ピヴォット（PV）

| 項目         | 内容                                   |
| ------------ | -------------------------------------- |
| ① 能力       | 接触耐性、体幹、位置取り               |
| ② テクニック | ポストワーク、スクリーン               |
| ③ シュート   | ターン、押し込み、ワンハンド           |
| ④ 弱点       | 可動域狭い、片側依存                   |
| ⑤ 強み       | 守備を引き付ける存在                   |
| ⑥ 役割       | 中央で守備を崩す起点                   |
| ⑦ 勘違い     | 「点だけ取ればいい」「動かなくていい」 |

## 指導的に重要なポイント

- **⑥役割**は「評価基準」
- **⑦勘違い**は「修正ポイント」
- ④弱点 × ⑦勘違い を潰す練習が最優先

この表は
✔ 選手への説明
✔ 自己評価シート
✔ 練習設計
すべてに使えます。

# ⑦ 勘違い → 正しい思考への言い換え表（ポジション別）

※「叱る表」ではなく
👉 **思考OSを書き換える表**として使ってください。

------

## ウイング（LW / RW）

| 勘違い（やりがち）       | 正しい思考（言い換え）       |
| ------------------------ | ---------------------------- |
| 点を取れれば仕事は終わり | **点＋幅＋速攻参加が役割**   |
| ボールを待つポジション   | **スペースを作るのが仕事**   |
| 角度がなければ無理       | **体勢とコースは自分で作る** |
| 守備はバックの仕事       | **最初のプレッシャーは自分** |

------

## バック（LB / RB）

| 勘違い                 | 正しい思考                     |
| ---------------------- | ------------------------------ |
| 強く投げれば入る       | **「速さ」より「早さと精度」** |
| 打てるなら全部打つ     | **打つことで味方を楽にする**   |
| 当たられるのは仕方ない | **当たられない位置取りがある** |
| 攻撃だけが評価         | **戻りと守備で信頼を得る**     |

------

## センターバック（CB）

| 勘違い               | 正しい思考                     |
| -------------------- | ------------------------------ |
| 自分が一番判断できる | **最善は「チームの選択」**     |
| 指示＝声を出すこと   | **動きで伝えるのも指示**       |
| 自分が点を取らないと | **点を取らせる方が価値が高い** |
| ミスは仕方ない       | **ミスの種類を選ぶ**           |

------

## ピヴォット（PV）

| 勘違い               | 正しい思考                   |
| -------------------- | ---------------------------- |
| 点を取れば評価される | **引き付けるだけで仕事成立** |
| 動かない方が楽       | **動くほど味方が楽になる**   |
| パスは来るもの       | **来る位置に入り直す**       |
| 押されるのは仕方ない | **体の向きで勝負は変わる**   |

------

### 指導での使い方（重要）

- 試合後ミーティングで
  👉「今のプレー、どの“勘違い”だった？」
- 練習前に
  👉 今日の**思考テーマを1つだけ**決める

これだけでプレーの質が変わります。
