### Q3. 練習と弱点/強みの対応

- ドリブル練習

```
[パス練習] [ドリブル練習] [シュート練習] [ウイング特訓] [バック特訓] [CB特訓] [PV特訓] ...
```

**案C) 現在のメニューで自動割り当て:**
```
パス練習 → CB, PVの弱点に影響
ドリブル練習 → LW, RW, LB, RBの弱点に影響
シュート練習 → 全ポジションのシュート弱点に影響
総合練習 → 全ポジションに均等に影響
```

### Q8. LW/RW、LB/RBの弱点/強み共有

**質問:** 同じ種別のポジション（LW/RW、LB/RB）は弱点/強みを共有しますか？

**案A) 種別で共有:**
```
LWの弱点を克服 → RWも同じ弱点が消える
（ウイング全体として管理）
```

**案B) ポジション別に独立:**
```
LWの弱点を克服 → LWのみ消える、RWは別途克服が必要
（6ポジション × 弱点数 = 合計弱点数が増える）
```

- [x] A) 種別で共有（ウイング/バック/CB/PVの4カテゴリで管理）

### Q9. 練習種類と弱点の対応関係

**質問:** Q3でA)を選択されましたが、具体的にどの練習がどの弱点に影響しますか？

**提案:**

| 練習 | 影響する弱点カテゴリ |
|------|---------------------|
| パス練習 | 判断系（判断遅れ、消極性、中央管理不足） |
| ドリブル練習 | 動作系（体勢不安定、可動域狭い、片側依存） |
| シュート練習 | シュート系（決定率の波、精度ムラ、リリース遅れ） |
| 総合練習 | 全カテゴリ均等（守備1on1不足、戻り遅い含む） |

- [x] A) 上記提案でOK
- [ ] B) 全練習が全弱点に均等に影響
