## 🎮 ローカルでの起動方法（Windows）

### 最も簡単な方法

1. `start-game.bat` をダブルクリック
2. ブラウザで `http://localhost:8000` を開く
3. プレイ開始！

### Pythonがない場合

[Python公式サイト](https://www.python.org/downloads/)からインストールしてください。
**重要**: インストール時に「Add Python to PATH」にチェックを入れること！

### その他の起動方法

#### Python手動起動
```bash
cd F:\LetsMakeHandballClub
python -m http.server 8000
```

#### Node.jsの場合
```bash
cd F:\LetsMakeHandballClub
npx http-server
```

#### VS Code Live Server
1. VS Codeでフォルダを開く
2. Live Server拡張機能をインストール
3. index.htmlを右クリック → "Open with Live Server"

## 🎯 ゲーム概要

### 目標
全国大会で6週間連続で勝ち続け、最強の「てぇでぇ's学園」を倒して優勝！

### 週間スケジュール
- **月〜金**: 練習日（5種類の練習メニューから選択）
- **土曜**: 全国大会の試合日
- **日曜**: 休養日（自動進行）

### 練習メニュー
- **パス練習**: パス能力強化
- **ドリブル練習**: ドリブル能力強化
- **シュート練習**: シュート能力強化
- **総合練習**: 全能力バランス強化
- **休養**（金曜のみ）: 次の試合で全能力+2

### 試合システム
1. 作戦立案フェーズで行動を設定
   - **パス**: ポジション選択（誰が→誰に）
     - LW（左ウイング）、RW（右ウイング）、CB（センターバック）
     - LB（左バック）、RB（右バック）、P（ピボット）

   - **ドリブル**: 段階的選択
     - 方向: 敵に向かって / 敵から逃げる / 右へ / 左へ
     - 距離: 短距離 / 中距離 / 長距離
     - 次の行動: 反対ドリブル / パス / コーナーシュート / 中央シュート

   - **シュート**: タイプ選択
     - コーナーめがけて丁寧にシュート（精度↑、威力↓）
     - とにかくど真ん中に最強シュート（精度↓、威力↑）

2. 実行フェーズでリアルタイム30fps再生
3. ズッキュン中学がシュート成功→1点、相手がボール奪取→相手1点
4. 先に5点取ったチームの勝利

### キャプテンシステム
- **性格** (3種類):
  - **パワハラ**: 成長1.5倍だが3週目以降ボイコット
  - **甘やかし**: 成長0.8倍で安定
  - **熱血**: 成長1.0倍でバランス型

- **方針** (4種類):
  - **アグレッシブ**: シュート重視
  - **慎重**: パス重視
  - **目立ちたがり**: ドリブル重視
  - **論理的**: 最も低い能力を重点強化

### エースシステム
- 試合に勝つたびに1人エースが誕生
- エースの能力は1.5倍
- 試合中は大きく表示される

### 地方別守備戦術
全国各地のチームが異なる守備戦術を使用：
- 北海道: 全員プレス
- 東北: マンツーマン
- 関東: ゾーンディフェンス
- 中部: プレスディフェンス
- 近畿: バランス型
- 中国: サイド守備重視
- 四国: センター守備重視
- 九州: 個人技重視
- K航拿: 完全守備（最強）

## 🎮 操作方法

### 基本操作
- すべてマウス/タッチで操作
- ボタンをクリック/タップして選択

### 試合での作戦立案
1. 「行動を選択」から選ぶ
2. パス/ドリブル/シュートのいずれか
3. 「追加」ボタンで作戦リストに追加
4. 必ず最後はシュートで終わる
5. 「試合開始」で実行

## 💾 セーブデータ

- **自動保存**: 各行動後に自動的にLocalStorageに保存
- **手動セーブ**: 不要（自動で保存されます）
- **続きから**: タイトル画面の「続きから」ボタン
- **データリセット**: ブラウザの開発者ツールでLocalStorageをクリア

### データをリセットする方法
1. ブラウザでF12キーを押す
2. 「Application」タブを開く
3. 「Local Storage」→ サイトURLを選択
4. 「handballGame」を削除
5. ページをリロード

## 🔧 カスタマイズ

### ゲームバランスの調整

`js/config.js` を編集することで、すべてのパラメータを変更できます：

```javascript
// 例: 初期能力値範囲を変更
export const INITIAL_STATS = {
    MIN: 5,    // 1から5に変更
    MAX: 15,   // 10から15に変更
    ALLOW_FLOAT: true
};

// 例: 勝利点数を変更
export const GAME_CONFIG = {
    POINTS_TO_WIN: 7,  // 5から7に変更
    // ...
};
```

### デバッグモードの有効化

`js/config.js` でデバッグモードを有効化：

```javascript
export const DEBUG_CONFIG = {
    ENABLED: true,  // falseからtrueに変更
    // ...
};
```

ブラウザコンソール（F12）で以下のコマンドが使えます：

```javascript
// 週をスキップ
window.debugGame.skipToWeek(3);

// 能力値を設定
window.debugGame.setStats(20, 20, 20);

// エースを追加
window.debugGame.addAces(5);

// 決勝へジャンプ
window.debugGame.advanceToFinals();

// 現在の状態を表示
window.debugGame.showState();

// ゲームをリセット
window.debugGame.resetGame();
```

## 🐛 トラブルシューティング

### ゲームが起動しない
- ブラウザのコンソール（F12）でエラーを確認
- ローカルサーバー経由で起動しているか確認
- ブラウザのキャッシュをクリア

### 画面が真っ白
- JavaScriptのエラーを確認（F12 → Console）
- ブラウザが最新版か確認
- プライベートブラウジングモードを試す

### セーブデータが消える
- ブラウザのCookieやLocalStorageが有効か確認
- プライベートモードで起動していないか確認
- ブラウザの容量制限に達していないか確認

### 試合が始まらない
- 作戦が正しく設定されているか確認
- 最後がシュートで終わっているか確認
- ブラウザコンソールでエラーを確認
